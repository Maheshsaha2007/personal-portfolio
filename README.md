# 🚀 Mahesh Saha — Personal Portfolio

## ✨ Features

- **Dark Neon Theme** — Deep blue palette with electric cyan (`#00f2ff`) accents
- **Glassmorphism UI** — Frosted glass containers with `backdrop-filter` blur effects
- **3D Tilt Interactions** — Mouse-responsive depth effects on cards and headings
- **Scroll Reveal Animations** — Sections animate smoothly into view on scroll
- **Responsive Design** — Optimized for desktop, tablet, and mobile screens

## 🚀 Getting Started

### View Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/personal-portfolio.git
   ```
2. Open `index.html` in your browser — no build tools or server required.

### Deploy to GitHub Pages
1. Push the repo to GitHub.
2. Go to **Settings → Pages**.
3. Set source to `main` branch and `/ (root)`.
4. Your site will be live at `https://YOUR_USERNAME.github.io/personal-portfolio/`.

## 🛠️ Customization

- **Change name/details**: Edit the text content in `index.html`
- **Change colors**: Update CSS custom properties in `:root` at the top of `style.css`
- **Add projects**: Duplicate a `.project-card` block in the Projects section
- **Add certificates**: Duplicate a `.cert-item` block in the Certificates section
